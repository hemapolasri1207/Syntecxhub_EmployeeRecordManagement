# Employee Record Management System (Java)

## 📌 Project Overview
This is a simple **Employee Record Management System** built using **Java Collections Framework**.  
The project demonstrates how to manage employee data efficiently using different collection types.

---

## 🚀 Features
- Add new employee records  
- Update existing employee details  
- Delete employee records  
- Display all employees  

---

## 🛠️ Technologies Used
- Java  
- Collections Framework (List, Set, Map)  

---

## 📂 Data Structures Used
- **ArrayList (List)** → Stores employee records in order  
- **HashSet (Set)** → Prevents duplicate employee IDs  
- **HashMap (Map)** → Enables fast search and updates  

---

## 💡 How It Works
1. User selects an operation from the menu  
2. Inputs employee details  
3. Data is stored and managed using collections  
4. System performs operations like add, update, delete, and display
